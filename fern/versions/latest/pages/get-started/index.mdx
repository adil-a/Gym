NeMo Gym is a library for building RL training environments for LLMs.

<Tip>
Already installed and have an API key? Jump straight to the [Quickstart](/latest/get-started/quickstart).
</Tip>

## Pick Your Path

| If you want to... | Start here | Best for |
| --- | --- | --- |
| **Run an evaluation end-to-end** | [Quickstart](/latest/get-started/quickstart) | Most users — installs, configures, and scores Blackjack in one page |
| **Understand the architecture first** | [About](/latest/about) → [Concepts](/latest/about/concepts) → [Quickstart](/latest/get-started/quickstart) | New to agent evaluation or LLM training |
| **Build a custom environment** | [Create Your First Environment](/latest/get-started/create-first-environment) | Comfortable with agentic RL; skips Quickstart and walks through env-building directly |

**What you need regardless of path:**

1. Python 3.12+ on a [verified configuration](/latest/get-started/prerequisites#verified-configurations).
2. An OpenAI-compatible inference endpoint and API key (or self-hosted vLLM — refer to [Configuration](/latest/get-started/configuration#do-you-need-an-api-key)).
3. Familiarity with the command line and basic Python.
