Generate synthetic multi-step user queries for the [Workplace Assistant](/latest/build-environments/real-world-environment) environment using [NeMo Data Designer](https://github.com/NVIDIA-NeMo/DataDesigner) — 27 tools across 6 databases, dual-level LLM judge filtering for quality, output in NeMo Gym JSONL format.

The pipeline produces task queries; rollout collection produces the actual model responses later. Simulated agent trajectories are used only for quality filtering and validation during generation.

<NavButton href="/latest/build-environments/real-world-environment" label="Back to Workplace Assistant" direction="back" />

---

## Pipeline Overview

1. Load tool schemas for the Workplace Assistant environment.
2. Generate realistic multi-step user queries with NeMo Data Designer.
3. Simulate agent trajectories (step-by-step tool-call solutions).
4. Apply dual-level LLM judge filtering for quality.
5. Export task data in NeMo Gym JSONL format.
